Exercise tracker

Fourth project from the certification Back End Development and APIs


From freeCodeCamp.org


Here we are working with backend javascript create a exercise tracker log where you can subscribe a user and with its personal id hash, you can post exercises for you to track in your app

Create a New User

POST /api/users

Add exercises

POST /api/users/:_id/exercises

GET user's exercise log: GET /api/users/:_id/logs?[from][&to][&limit]

[ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
